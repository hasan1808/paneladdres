# پنل مدیریت سرورهای VPN

یک سیستم مدیریت سرور VPN شامل صفحه دانلود عمومی و پنل ادمین برای مدیریت سرورهای OpenVPN، Cisco و L2TP.

## ⚠️ نکات امنیتی مهم

### ۱. تغییر رمز ادمین
پس از نصب، حتماً رمز پیش‌فرض ادمین را تغییر دهید:
1. وارد پنل ادمین شوید (رمز پیش‌فرض: `admin123`)
2. به بخش **تنظیمات** بروید
3. رمز عبور جدید را وارد کرده و ذخیره کنید

### ۲. تغییر نام فایل ادمین
برای افزایش امنیت، حتماً نام فایل `admin.html` را تغییر دهید تا مسیر پنل مدیریت مخفی شود:
- فایل `admin.html` را به یک نام دلخواه (مثلاً `panel-xyz.html`) تغییر نام دهید
- هیچکس بدون دانستن نام جدید به پنل دسترسی نخواهد داشت

## امکانات

### صفحه اصلی (`index.html`)
- نمایش سرورها در سه تب: OpenVPN، Cisco، L2TP
- گروه‌بندی سرورها بر اساس منطقه (اروپا، آسیا، آمریکا، خاورمیانه و آفریقا)
- دانلود مستقیم فایل کانفیگ OpenVPN (`.ovpn`)
- کپی آدرس سرورهای Cisco و L2TP
- نمایش تنظیمات Cisco AnyConnect و L2TP/IPsec
- نمایش پرچم کشورها با تصویر واقعی

### پنل ادمین (`admin.html`)
- صفحه ورود با رمز عبور
- داشبورد با آمار سرورها
- مدیریت سرورها (افزودن، ویرایش، حذف)
- پشتیبانی از هر سه پروتکل: OpenVPN، Cisco، L2TP
- آپلود فایل `.ovpn` برای سرورهای OpenVPN (از طریق `upload.php`)
- حذف فایل‌های `.ovpn` (از طریق `delete.php`)
- تنظیمات (تغییر رمز، نام کاربری و رمز Cisco، PSK و اطلاعات L2TP)
- برون‌بری و درون‌بری داده‌ها (JSON)
- ریست کردن اطلاعات
- نمایش/عدم نمایش سرورهای غیرفعال
- طراحی واکنش‌گرا (Responsive)
- منوی جانبی قابل جمع شدن در موبایل

## نحوه نصب

1. فایل‌های پروژه را در هاست (مثلاً `public_html` یا `htdocs`) آپلود کنید
2. فایل‌های `.ovpn` را در پوشه `file/` قرار دهید
3. مرورگر را باز کرده و به آدرس پنل ادمین بروید
4. با رمز پیش‌فرض `admin123` وارد شوید
5. رمز را تغییر دهید و نام فایل ادمین را تغییر دهید

## ساختار پوشه‌ها

```
paneladdres/
├── index.html          # صفحه اصلی دانلود سرورها
├── admin.html          # پنل مدیریت (تغییر نام دهید)
├── upload.php          # آپلود فایل‌های .ovpn
├── delete.php          # حذف فایل‌های .ovpn
├── file/               # فایل‌های کانفیگ .ovpn
└── README.md           # این فایل
```

## ذخیره‌سازی

تمام اطلاعات سرورها و تنظیمات در `localStorage` مرورگر ذخیره می‌شوند. برای انتقال داده‌ها از قابلیت برون‌بری (Export) در پنل ادمین استفاده کنید.

## پروتکل‌ها

| پروتکل | توضیح | فرمت آدرس/فایل |
|--------|-------|----------------|
| OpenVPN | دانلود فایل کانفیگ `.ovpn` | فایل `.ovpn` در پوشه `file/` |
| Cisco AnyConnect | کپی آدرس سرور + تنظیمات AnyConnect | آدرس سرور |
| L2TP/IPsec | کپی آدرس سرور + تنظیمات L2TP | آدرس سرور |
